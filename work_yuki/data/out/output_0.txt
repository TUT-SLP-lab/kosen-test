二分探索のプログラムをC言語で書いてください。標準入力からソート済みの配列Aの長さNと要素A_1...A_N、探索する値Bが与えられます。BがAに存在する場合はインデックス、存在しない場合は-1を標準出力に出力してください。回答以外は標準出力には出力してはいけません。

以下が、C言語での二分探索プログラムです。ソートされた配列Aの長さN、要素A_1...A_N、および検索する値Bが標準入力から与えられます。BがAに存在する場合はそのインデックスを出力し、存在しない場合は-1を出力します。答え以外は標準出力に何も出力しないでください。

```c
#include <stdio.h>

int binarySearch(int arr[], int size, int target) {
    int left = 0;
    int right = size - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (arr[mid] == target)
            return mid;
        else if (arr[mid] < target)
            left = mid + 1;
        else
            right = mid - 1;
    }
    
    return -1;
}

int main() {
    int N, B;
    scanf("%d", &N);
    int A[N];
    for (int i = 0; i < N; i++) {
        scanf("%d", &A[i]);
    }
    scanf("%d", &B);
    
    int result = binarySearch(A, N, B);
    printf("%d\n", result);
    
    return 0;
}
```
